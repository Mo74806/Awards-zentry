## Getting Started

### Prerequisites

- Node.js v16+  
- npm or yarn

---

### Installation

```bash
git clone https://github.com/Mo74806/Awards-zentry.git
cd Awards-zentry
npm install
```

---

### Development

Start the dev server:

```bash
npm run dev
```

Visit `http://localhost:3000` to view the site.
